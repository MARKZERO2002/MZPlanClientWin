# 介绍
一个基于Qt C++编写的Windows程序，用于帮助用户管理自己的日程、计划。

clone,zip，下载后是一个开发环境的Qt qmake程序。

轻量级、方便、没有任务栏、可选择固定到桌面或隐藏、可快捷键呼出(自定义)
